# version

Show version and build information.

```bash
rsdedup version
```

Example output:

```
rsdedup 0.1.0 by Mark Veltzer <mark.veltzer@gmail.com>
GIT_DESCRIBE: v0.1.0
GIT_SHA: abc123def456
GIT_BRANCH: master
GIT_DIRTY: false
RUSTC_SEMVER: 1.94.0
RUST_EDITION: 2024
BUILD_TIMESTAMP: 2026-03-24 01:30:35
```
